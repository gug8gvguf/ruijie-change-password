# 锐捷云课堂脱控教程

想脱控需要对终端和win虚拟机都设置

win方法
1下载脚本并运行
2点击云课堂右上倒三角，选择关闭软件
3输入你改的密码（没改默认 ruijie.com）点击确定
4此时已关闭win中云课堂，部分电脑有画面残留，不用在意



安卓终端脱离广播方法
1拔网线
2按下alt shift f7(一起)
3输入密码（默认 ruijie.com）,改密码只能去服务器网页操作
4进入桌面后点击设置图标，左边找到开发者选项，右边拉最下面，将后台进程限制从标准限制改为不允许后台进程
5按esc多次返回桌面，点中间圆，再点云课堂，回到开始界面
6插网线，等待一会，安卓广播画面没了，会进入win虚拟机界面

win不脱有你正在收看广播画面，安卓不脱会有教师机画面
